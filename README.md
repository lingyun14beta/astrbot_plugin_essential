# AstrBot Essential

为 [AstrBot](https://github.com/AstrBotDevs/AstrBot) 提供基本的娱乐功能。

## 支持功能

- 以图搜番（[trace.moe](https://trace.moe)）
- 随机动漫图片
- 喜报 / 悲报生成
- Minecraft 服务器查询
- 一言
- 今天吃什么
- EPIC 喜加一
- 早安 / 晚安（群内睡眠记录）

## 安装

```
plugin i https://github.com/Soulter/astrbot_plugin_essential
```

## 使用方法

| 命令 | 说明 |
|---|---|
| `/搜番` | 以图搜番。发送命令后，在 30 秒内发送一张图片即可识别 |
| `/moe` | 随机动漫图片 |
| `/喜报 <内容>` | 生成喜报图片 |
| `/悲报 <内容>` | 生成悲报图片 |
| `/mcs <服务器地址>` | 查询 Minecraft 服务器状态 |
| `/一言` | 随机一句话 |
| `/今天吃什么` | 随机选择今天吃什么 |
| `/今天吃什么 添加 食物1 食物2 ...` | 添加食物到列表 |
| `/今天吃什么 删除 食物1 食物2 ...` | 从列表删除食物 |
| `/喜加一` | 查询 EPIC 当前及即将免费的游戏 |
| 早安 | 记录起床时间，显示昨晚睡眠时长 |
| 晚安 | 记录入睡时间，显示今天群内第几个睡觉 |

> 早安 / 晚安 直接发送文字触发，不需要加 `/`。

> **从旧版插件升级**：早晚安历史数据会自动迁移。若旧版开启了隔离会话，升级后历史睡眠记录将丢失，之后的数据正常记录。

## 管理员命令

| 命令 | 说明 |
|---|---|
| `/清除缓存` | 清除喜报、悲报、moe 的图片缓存文件 |

## 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `report_font_size` | int | 65 | 喜报 / 悲报的字体大小 |
